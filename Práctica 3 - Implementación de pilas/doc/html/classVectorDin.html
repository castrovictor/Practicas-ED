template<class T>
class VectorDin< T >

T.D.A. VectorDinamico.

Una instancia c del tipo de datos abstracto VectorDin es un objeto con 3 campo, un vector dinámico de un objeto template T, y dos int, uno para el número de objetos utiles en el vector y otro con el numero de posiciones reservadas
